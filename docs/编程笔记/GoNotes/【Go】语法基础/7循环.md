# 7 循环

## 循环方式1
```go
nums := []int{1, 2, 3, 4, 5, 6}

for i := 0; i < len(nums); i++ {
		fmt.Println(i)
	}
```

格式还是`for`后面加`i`的变量的初始循环值，循环的长度，i++。

## 循环方式2
```go
a, b := 1, 5
for a < b {
		fmt.Println(a)
		a++
	}
```

类似于while，和上面的循环方式1相比，省略了赋值和++

## 循环方式3
```go
for index, value := range nums {
		fmt.Printf("key: %v , value: %v
    \n", index, value)
	}
```

迭代的方式进行循环，这里的`index`为数组的下标，`value`为数组里面的值，可以忽略下标`index`或者值`value`，用下划线带代替，这样就可以避免引入无关的变量定义。

这种循环的定义方式类似于python的`for key in nums[]`。

## 死循环写法
```go
for{
}
```

## 案例练习
```go
package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
)

func convertToBin(n int) string{
	result := ""
	for ; n>0 ; n /= 2{
		lsb := n%2
		result = strconv.Itoa(lsb) + result //每次的余数加到字符串的前面
	}
	return result
}
func printFile(filename string){
	file,err := os.Open(filename)
	if err != nil{
		panic(err)
	}
	scanner := bufio.NewScanner(file) //一行行读
	for scanner.Scan(){//直接for一个条件就可，没有起始条件和结束条件
		fmt.Println(scanner.Text())
	}
}
func forever(){
	for { //死循环，go并发时候经常死循环
		fmt.Println("abc")
	}
}
func main() {
	fmt.Println(
		convertToBin(5),//101
		convertToBin(13),//1101
		)
	printFile("abc.txt")
	forever()
}

```
