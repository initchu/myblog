# 9 工厂模式

由于go语言中没有[构造函数](https://baike.baidu.com/item/%E6%9E%84%E9%80%A0%E5%87%BD%E6%95%B0/7478717?fr=aladdin)，所以采用工厂模式来解决这个问题。

针对model包中的结构体User，我们想要创建一个实例，创建的时候需要进行直接创建`<font style="color:#1890FF;">model.User</font>`。

但是如果，这个User结构体的首字母是小写的，那我们应该如何创建这个结构体的变量呢？

工厂模式就是为了解决这个问题而存在的。

我们在model包中添加`<font style="color:#1890FF;">NewUser</font>`代码以实现工厂模式：

```go
package model
type user struct{
    Name string
    Sex string
}
func NewUser(n string, s string) *user {
    return &user{
        Name: n,
        Sex: s,
    }
}

```

```go
package main
func main(){
    var user1 = model.NewUser("Bob","男")
}
```
