# 2 数组

### 与其它语言区分
GO语言一种独有的结构，可以用来遍历访问数组的元素。for-range结构遍历：

```go
for index,value := range array02{..}
```

### 说明
1. 第一个返回值index是数组的下标；
2. 第二个value是在该下标位置的值；
3. 它们都是仅在for循环内部可见的局部变量；
4. 遍历数组元素的时候，如果不想使用下标index，可以直接把下标index标为下划线；
5. index和value的名称不是固定的，即程序员可以自行指定，一般命名为index和value；



### 其他注意细节
1. 数组是多个相同类型数据的组合，一个数组一旦声明，其长度是固定的，不能动态变化；
2. [ ]int 这个不是数组，是切片；
3. 元素可以是任何数据类型，但是不能混用；
4. 数组如果没赋值，是有默认值的；
5.  使用数组的步骤：开空间->赋值->使用；
6. 下标从0开始；
7. 不能越界
8. 值类型，值传递；
9. 如果想在其他函数中，修改原来的数组，可以使用指针 arr *[3]int ；
10. 长度是数组的一部分，传递参数时候，要考虑数组的长度。 



### 实例练习
```go
package main
import "fmt"

func printArray(arr [5]int){
	arr[0] = 100 //值类型没办法改变
	for i,v := range arr{
		fmt.Println(i,v)
	}
}
func main() {
	//数量写在类型的前面
	var arr1 [5]int
	arr2 := [3]int{1,3,5}
	arr3 := [...]int{2,4,5,6,8}
	var grid [4][5]int
	fmt.Println(arr1,arr2,arr3)
	fmt.Println(grid)
	//遍历方法1
	for i := range arr3{
		fmt.Println(arr3[i])
		//fmt.Println(i,v)
	}
	//way2
	for i,v := range arr3{
		fmt.Println(i,v)
	}
	//way3
	for _,v := range arr3{
		fmt.Println(v)
	}
	fmt.Println("PrintArray(arr1)")
	printArray(arr1)
	//printArray(arr2) 不行
	fmt.Println("PrintArray(arr3)")
	printArray(arr3)
//数组是值类型（拷贝的值）
//go一般不直接使用数组，使用切片
}

```
