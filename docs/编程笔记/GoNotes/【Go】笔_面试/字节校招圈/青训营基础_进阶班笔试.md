## top 命令给出的指标，正确的是
A：Si表示系统处理硬中断 CPU 占用比例

B：VIRT 代表进程实际使用物理内存量

C：load average 越高说明系统压力越小

D：zombie 进程数量代表以及退出待父进程回收的进程数量

**答案&&解析**：

:::success
 选择D。

A，si：软中断（模式切换）

B，VIRT：进程使用的虚拟内存总量

C，load average 越高压力越大

D，表示僵尸进程数，僵尸进程是当子进程比父进程先结束，而父进程又没有回收子进程，释放子进程占用的资源，此时子进程将成为一个僵尸进程

:::

## 编程题 1（ACM 模式）
有效 IP 地址正好由四个整数（每个整数位于 0 到 255 之间，且不能含有前导 0）组成，整数之间用`.`分隔。

例如：

“0.1.2.201” 和“192.168.1.1” 是有效地址，

“0.011.255.245”、“192.168@1.1”、“192.169.1.312”是无效地址。

给定一个字符串 s，非数字得字符可替换为任意不包含在本字符串的数字，同样得字符只能替换为同样的数字，用以表示一个 IP 地址，返回所有可能的有效 IP 地址，并按照数字默认顺序排序，`.`应该排在数字前面，例如 1.23.4.5 应该在 12.3.4.5 前面。

注意：不能重新排序或者删除 s 中的任何数字。
