# 单双链表的实现

## 单链表
使用带head头的单向链表实现，完成增删改查操作。

package main
type LinkNode struct{
    value int
    next *LinkNode
}
//第一种插入方法，直接插入到链表结尾
func InsertNode(head *LinkNode, newNode *LinkNode){
    temp := head
    for{
        if temp == nil{
            break
        }
        temp = temp.next //指针后移
    }
    temp.next = newNode
}
