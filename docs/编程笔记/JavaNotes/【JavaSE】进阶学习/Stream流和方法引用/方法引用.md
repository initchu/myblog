# 方法引用
+ 把已经有的方法拿过来用，当做函数式接口中抽象方法的方法体
+ 例子：假设有如下的方法

```java
Arrays.sort(arr, new Comparator<Integer>(){
    @Override
    public int compare(Integer o1, Integer o2){
        return o2 - o1;
    }
});
```

1. 引用处必须是函数式接口。
