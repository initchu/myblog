# 包的命名
## 命名规则
只能包含数组、字母、下划线、小圆点。

不能用数字开头，不能是关键字或者保留字。

## 命名规范
一般是小写字母+小圆点。

com.公司名字.项目名字.业务模块名

比如：com.javastudy.demo.model、com.javastudy.demo.controller；

# import
`import java.util.Scanner` 表示只会引入 java.util 包下的 Scanner

`import java.util.*` 表示将 java.util 包下的所有类都引入 

常用的包：

`java.lang.*`基本包，默认引入，不需要再引入。

`java.util.*`util 包，系统提供的工具包，工具类，使用 Scanner

`java.net.*`网络包，网络开发

`java.awt.*`GUI 包
